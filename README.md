Nele contém códigos que utilizam:

- Structs;
- Matrizes;
- Vetores; 
- Repetições;
- Strings;
- Arquivos;
- Entre outros.

####A pasta LAB é referente aos códigos feitos na aula de laboratório, enquanto a pasta SONED é referente aos códigos feitos na aula do professor do curso.
